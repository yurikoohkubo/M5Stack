PL_ImageConstructor Manual
一括伸縮
概要
画像データの拡大や縮小を行います。
サムネールを作成したりするのに利用できます。
各種コントロール
  • 縦横比維持
    元のサイズの縦横の比率を維持します。

    • 比率
      パーセンテージで指定します。100にすると変化しません。

    • 横幅
      横幅を指定したピクセルにして伸縮します。
      高さの算出は { 元の画像の高さ * 指定した横幅 / 元の画像の横幅 } です。

    • 高さ
      縦幅を指定したピクセルにして伸縮します。
      横幅の算出は { 元の画像の横幅 * 指定した高さ / 元の画像の高さ } です。


  • 縦横個別指定
    横幅と高さをここにサイズ指定して伸縮します。


  • 補間法
    伸縮時に行われる画質を指定します。

    • バイキュービック
      周囲のピクセルの情報をバイリニア以上に取得し補間します。

    • バイリニア
      周囲のピクセルの情報を取得し補間します。

    • ニアレストネイバー
      荒く処理します。
      ドット絵を等倍にしたいなど余計な補間を行いたくない場合に最適です。


  • ディレクトリの構造を維持
    登録されたファイルのフォルダ構成再現して出力します。
